Help / Rules

Rules

1. No illegal content
2. No spam or flooding
3. Image required for new threads
4. Text limited to 2000 characters
5. Rate limits:
   - New thread: 1 per 30 seconds
   - Reply: 1 per 10 seconds
   - General: 60 requests per minute
6. All data is ephemeral - do not rely on permanence

How to Use

Creating a Thread:
1. Select a board from the home page
2. Click [New Thread]
3. Add an image (required)
4. Write your text
5. Optional: Add a name

Posting a Reply:
1. Open a thread
2. Scroll to reply form
3. Write your text (required)
4. Optional: Add image and/or name

Tripcodes:
Enter "name#password" to generate a unique identifier
Example: "Lain#wired" → "Lain ◆AbCdEfGhIj"
The password is hashed - impossible to reverse

Philosophy

"All data is ephemeral. Threads vanish when memory fades."

This is not a bug - it is the core design.
Nothing here is permanent.
Nothing here is archived.
When it's gone, it's gone forever.

Embrace the impermanence.
